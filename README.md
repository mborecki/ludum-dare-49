Game for Ludum dare 49.

# Framework

* Phaser 3.1.2

# Dependencies

For audio to work you need installed ffmped on your system.
Download from https://www.ffmpeg.org/download.html


------------------------------------------------------------------------------------------

# How to use?

## Setup

```sh
yarn
```

## Developing

```sh
yarn server
```

## Build

```sh
yarn build
```
